Project Description:

DevDash is a simple developer dashboard built using HTML, CSS, and JavaScript.
The dashboard displays a developer profile, projects section, a todo list for managing tasks, and a live digital clock. It also includes responsive design and interactive UI features.

Features:

* Developer profile section
* Projects showcase
* Todo list with add and delete functionality
* Live digital clock
* Dynamic footer year
* Responsive layout

How to Run the Project:

1. Download or clone the repository
2. Open the project folder
3. Open **index.html** in your browser

No additional installation is required.

GitHub Pages Link: https://github.com/sania-elsa/pagesoft-frontend-intern-assignment

### Development Decisions

Todo List Implementation:

The todo list was implemented using JavaScript and browser localStorage.
Tasks are stored in localStorage so that they persist even after refreshing the page.
Users can add and delete tasks dynamically.

Live Digital Clock:

The live clock was implemented using JavaScript's Date object.
A function updates the time every second using setInterval(), ensuring the displayed time stays accurate.

Responsive Design:

CSS Grid and Flexbox were used to make the layout responsive.
Media queries ensure the dashboard works well on both desktop and mobile devices.
